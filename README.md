# Uma's Engineering Wiki

Welcome to Uma's Engineering Wiki! This repository serves as a comprehensive, beginner-friendly guide to modern software engineering tools and practices, focusing on getting you up to speed quickly and scalably.

Whether you're just starting out or looking to formalize your knowledge, this wiki is structured to take you from basic version control to advanced agentic coding workflows using Bazel, Docker, and local language models (Ollama).
