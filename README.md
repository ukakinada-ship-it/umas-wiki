# Uma's Engineering Wiki

Welcome! This wiki is a comprehensive, beginner-friendly guide to the modern engineering toolkit — from version control basics all the way to running local AI models and building scalable containerized environments.

Use the **sidebar on the right** to navigate between sections.

## What's Covered

| # | Section | What you'll learn |
|---|---------|-------------------|
| 1 | **Git & GitHub** | Version control, the git CLI, pull requests, diffs, and the `gh` CLI |
| 2 | **Agentic Coding** | Working with Antigravity, effective prompting, structuring repos for AI agents, AGENTS.md, skills, the full git workflow, and reviewing agent code |
| 3 | **Bazel** | Why Bazel exists, how to structure a workspace, and essential commands |
| 4 | **Ollama** | Running LLMs locally, model selection, and compute requirements |
| 5 | **Docker + Bazel** | Dev containers, running Bazel inside Docker, Python toolchains, and a full worked multi-Python example |
| 6 | **OpenClaw Integration** | What OpenClaw is, how to set it up, and how to utilize its rich ecosystem of 50+ built-in skills to supercharge your workspace |

## Recommended Reading Order

If you're brand new to all of these topics, read the sections top-to-bottom. Each one builds on the previous.

If you're already comfortable with Git, jump straight to **Agentic Coding** and work forward from there — that's the fastest path to getting started with scalable AI-assisted development.
